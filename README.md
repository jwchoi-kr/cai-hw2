# CAI HW2

Basic homework scaffold for a travel planning chatbot that detects user intent and pulls supporting location data.

The app interprets natural-language travel requests, classifies user intent, and fetches destinations or points of interest from external APIs to help craft itineraries quickly.

## Installation

1. Clone the repository.
2. Create and activate a virtual environment (optional but recommended).
3. Install the dependencies.

```bash
git clone <repo-url>
cd hw2
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```
